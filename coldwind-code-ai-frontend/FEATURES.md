# 功能演示说明

本文档详细说明了 Coldwind Code AI Frontend 项目的各项功能和使用方法。

## 🏠 主页功能演示

### 1. 网站标题区域

- **标题**：显示"一句话🐱呈所想"
- **副标题**：显示"与 AI 对话轻松创建应用和网站"
- **设计**：渐变背景，动画图标，现代化设计

### 2. 提示词输入区域

- **输入框**：用户可以输入应用描述
- **创建按钮**：点击后创建新应用并跳转到对话页面
- **快捷模板**：提供常用应用类型的快捷按钮
  - 波普风电商页面
  - 企业网站
  - 电商运营后台
  - 暗黑话题社区

### 3. 我的作品区域（登录用户可见）

- **应用列表**：分页显示用户创建的应用
- **搜索功能**：支持按应用名称搜索
- **操作按钮**：
  - 👁️ 查看详情：跳转到应用对话页面
  - ✏️ 编辑应用：跳转到应用编辑页面
  - 🗑️ 删除应用：删除应用（需确认）
- **分页控制**：支持分页浏览，每页最多20个应用

### 4. 精选案例区域

- **应用展示**：分页显示精选应用
- **搜索功能**：支持按应用名称搜索
- **应用信息**：显示应用名称、创建者、创建时间
- **操作按钮**：👁️ 查看详情

## 💬 应用对话页面演示

### 1. 顶部栏

- **应用信息**：显示应用名称和应用类型
- **操作按钮**：
  - 查看部署：如果已部署，显示部署链接
  - 部署：部署应用到服务器

### 2. 左侧对话区域

- **对话历史**：显示用户和AI的对话记录
- **消息样式**：
  - 用户消息：蓝色背景，右对齐
  - AI消息：灰色背景，左对齐，支持代码显示
- **输入框**：用户可以输入新的提示词
- **发送按钮**：发送消息给AI

### 3. 右侧网页展示区域

- **状态显示**：显示生成状态（等待生成/生成中/生成完成）
- **代码预览**：显示AI生成的代码内容
- **实时更新**：支持SSE流式响应，实时显示生成内容

## ✏️ 应用编辑页面演示

### 1. 页面头部

- **返回按钮**：返回上一页
- **页面标题**：显示"编辑应用"
- **保存按钮**：保存修改的应用信息

### 2. 应用信息展示

- **应用封面**：显示应用封面图片或占位符
- **基本信息**：显示应用名称、类型、创建者、时间等
- **操作历史**：时间线显示应用的创建、更新、部署记录

### 3. 编辑表单

- **应用名称**：可编辑的应用名称
- **应用封面**：仅管理员可见，可设置封面图片URL
- **优先级**：仅管理员可见，可设置应用优先级
- **初始提示词**：只读显示，不可编辑

### 4. 权限控制

- **普通用户**：只能编辑自己的应用，且只能编辑应用名称
- **管理员**：可以编辑所有应用的所有字段

## 🔧 应用管理页面演示（管理员专用）

### 1. 页面头部

- **页面标题**：显示"应用管理"
- **页面说明**：显示"管理系统中的所有应用"
- **重置按钮**：重置搜索条件

### 2. 搜索功能

- **应用名称**：按应用名称搜索
- **创建者**：按创建者姓名搜索
- **应用类型**：按应用类型搜索
- **用户角色**：按用户角色筛选（普通用户/管理员）

### 3. 应用列表

- **表格展示**：分页显示所有应用
- **列信息**：
  - 封面：应用封面图片
  - 应用名称：应用名称和类型
  - 创建者：创建者姓名和角色标签
  - 状态：应用状态（正常/精选/已部署）
  - 创建时间：应用创建时间
  - 操作：各种操作按钮

### 4. 操作功能

- **查看详情**：跳转到应用对话页面
- **编辑应用**：跳转到应用编辑页面
- **设置精选**：设置应用为精选应用（优先级99）
- **删除应用**：删除应用（需确认）

## 👥 用户管理功能

### 1. 用户登录

- **登录表单**：用户名/密码输入
- **表单验证**：必填字段验证
- **登录状态**：登录成功后显示用户信息

### 2. 用户注册

- **注册表单**：用户名、密码、确认密码
- **密码验证**：两次密码输入必须一致
- **注册成功**：注册成功后跳转到登录页面

### 3. 权限管理

- **普通用户**：只能访问主页、应用对话、应用编辑等基本功能
- **管理员**：可以访问所有功能，包括用户管理和应用管理

## 📱 使用流程

### 1. 创建应用流程

1. 在主页输入应用描述
2. 点击"创建应用"按钮
3. 系统创建应用并跳转到对话页面
4. 自动发送初始提示词给AI
5. AI开始生成代码，实时显示进度
6. 生成完成后可以继续对话优化
7. 满意后可以部署应用

### 2. 编辑应用流程

1. 在主页点击应用的"编辑"按钮
2. 跳转到应用编辑页面
3. 修改应用信息（名称、封面、优先级等）
4. 点击"保存"按钮保存修改
5. 返回上一页或继续编辑

### 3. 管理应用流程（管理员）

1. 在导航菜单点击"应用管理"
2. 进入应用管理页面
3. 使用搜索功能筛选应用
4. 对应用执行各种操作（查看、编辑、删除、设置精选）
5. 管理应用状态和优先级

## 🔒 安全特性

### 1. 权限控制

- **路由守卫**：未登录用户无法访问受保护页面
- **API权限**：后端API接口权限验证
- **角色控制**：管理员和普通用户权限分离

### 2. 数据验证

- **前端验证**：表单输入验证和格式检查
- **后端验证**：服务器端数据验证和清理
- **类型安全**：TypeScript类型检查防止运行时错误

### 3. 错误处理

- **网络错误**：网络请求失败的处理和重试
- **业务错误**：业务逻辑错误的友好提示
- **异常捕获**：全局异常捕获和错误日志
